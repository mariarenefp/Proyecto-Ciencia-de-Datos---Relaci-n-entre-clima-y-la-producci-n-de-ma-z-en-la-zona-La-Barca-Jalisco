# Proyecto-Ciencia-de-Datos---Relaci-n-entre-clima-y-la-producci-n-de-ma-z-en-la-zona-La-Barca-Jalisco
Proyecto de ciencia de datos que analiza la relación entre variables climáticas y la producción de maíz en la Ciénega de Jalisco. Incluye limpieza, visualización y modelos de regresión, junto con el ensayo final y la presentación del análisis.
